## ⚡ Super Quick Start (5 Minutes)

### Step 1: Install Software (One-time only)

**Need Python?**
- Download: https://www.python.org/downloads/
- Install and CHECK "Add Python to PATH"

**Need Node.js?**
- Download: https://nodejs.org/
- Install with default settings

### Step 2: Automated Setup

**Double-click:** `setup.bat`

This will install everything automatically!

### Step 3: Start the App

**Double-click:** `start.bat`

This opens 2 windows:
- Backend Server (Python)
- Frontend Server (React)

### Step 4: Open Browser

Go to: **http://localhost:5173**

**That's it! You're done! 🎉**

## 🆘 Something Not Working?

### "Python not found"
- Reinstall Python
- Check "Add to PATH" during installation
- Restart computer

### "npm not found"
- Reinstall Node.js
- Restart computer

### "Cannot connect to backend"
- Make sure backend window is open
- Check if it says "Uvicorn running on http://0.0.0.0:8000"

### "Port already in use"
- Close other programs
- Restart computer
