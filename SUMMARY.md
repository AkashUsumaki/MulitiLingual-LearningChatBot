## 🐛 Troubleshooting

### Common Issues

**"Python not found"**
- Install Python 3.8+
- Add to PATH

**"npm not found"**
- Install Node.js 16+
- Restart terminal

**"Cannot connect to backend"**
- Ensure backend is running
- Check port 8000

**"CORS error"**
- Backend CORS is configured
- Check API_URL in App.jsx
